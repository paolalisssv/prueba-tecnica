# prueba-t-cnica
#### Tecnologias 

- Frontend VueJs
- Backend NodeJs
- BD Mongodb

####  Installation backend

```bash
cd backend
npm i
```

####  Run Backend

```bash
npm run start
```

####  Installation frontend

```bash
cd frontend
npm i
```

####  Run Backend

```bash
npm run serve
```
